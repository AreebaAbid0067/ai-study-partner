# AI Study Partner

An intelligent study assistant powered by AI to help students with their academic questions.

## Project Structure

- **frontend/**: Streamlit-based user interface
- **backend/**: FastAPI backend server
- **requirements.txt**: Python dependencies

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Run backend: `uvicorn backend.main:app --reload`
3. Run frontend: `streamlit run frontend/app.py`

## Features

- Q&A system for various subjects
- User authentication
- Study history tracking
